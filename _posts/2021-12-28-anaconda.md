## 1. 가상 환경 생성
```
conda create -n 가상환경이름
```

## 2. 가상 환경 삭제
```
conda env remove -n 가상환경이름 
```

## 3. 가상 환경에 설치된 패키지 확인
``` 
conda list
```

## 4. 패키지 설치
```
conda install 패키지이름
```

## 5. 패키지 삭제
```
conda remove 패키지이름
```
